# 接口文档

### 1.成绩添加

请求URL：

- /score/addScore

请求方式：

- post

返回数据：

| 参数名  | 类型    | 长度 | 必选 | 说明                       |
| ------- | ------- | ---- | ---- | -------------------------- |
| subject | varchar | 11   | 是   | 科目名                     |
| score   | float   | 10   | 是   | 成绩(允许小数)             |
| type    | varchar | 12   | 是   | 期中、期末、月考，单元考试 |
| t_date  | date    | 0    | 是   | 考试日期                   |
